# 🏏 CricLens – Cricket Intelligence Web App

## 📌 Project Overview
CricLens is an cricket analysis web application that helps users understand player performance based on pitch and ground conditions.

## 🚀 Features
- Player search with autocomplete
- Ground-based pitch selection
- Performance score system
- Player strengths and weaknesses analysis
- Ground pitch behavior insights
- Intelligent explanation system

## 🧠 Unique Element
### Player × Pitch Intelligence Engine
This feature analyzes a player’s performance based on:
- Pitch type (spin, pace, flat)
- Ground conditions
- Player strengths and weaknesses

It generates a smart explanation of WHY a player performs well or struggles on a particular ground.

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- jQuery

## 🎯 How to Run
1. Download project
2. Open index.html in browser
3. Use Live Server (recommended)
